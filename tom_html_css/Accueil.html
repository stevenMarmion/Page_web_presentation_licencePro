Dernières informations

Journée Porte Ouverte du département informatique à Orléans !

Cette journée porte ouverte vous permettra de découvrir en quoi consiste le BUT Informatique, et d’y rencontrer des étudiants qui pourront vous faire découvrir les différentes matières à travers des présentations ou bien même des activités !

Parcoursup !

Parcoursup ouvre ses portes pour la 6ième années consécutives ! N’oubliez pas de vous inscrire et de consulter les différentes formations disponibles !

Bienvenue !

Ce site a pour objectif de vous faire découvrir les différents chemins que vous pouvez emprunter après avoir obtenu votre BUT Informatique. Cela peut aller d’une licence professionnelle, une école d’ingénieurs ou directement, une insertion dans le monde du travail.

Directement après le BUT ?

Que faire après un BUT Informatique. Vous vous êtes sûrement déjà posé la question. Et bien cette page est faites pour vous ! Dans celle-ci vous pourrez y retrouver les informations dont vous aurez besoin pour savoir quel poste et métier sont disponibles directement après un BUT informatique. Cliquez sur l'image pour y accéder.

Une licence professionnelle ?

La licence professionnelle vous tente ? Dans cette page vous pourrez y retrouver les différentes possibilités à venir qui s’offrent à vous si vous souhaitez vous orienter vers une licence en informatique. Cliquez sur l'image pour y accéder.

Master en Informatique ?

Le master en informatique ? Diplôme réputer dans le domaine ! Mais vers quel poste ou métier peut-il réellement vous amener ? Dans cette page, vous pourrez y retrouver toutes les informations dont vous avez besoin. Cliquez sur l'image pour y accéder.

Écoles d'ingénieurs en Informatique ?

Réputer pour le titre qu’elles délivrent, les écoles d’ingénieurs peuvent vous mener vers une multitude de métier, mais lequel est fait pour vous ? Dans cette page, vous y retrouveraient toutes les informations qui pourront vous aider à faire votre choix ! Cliquez sur l'image pour y accéder.

Nous suivre sur les réseaux !
